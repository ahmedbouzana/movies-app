# Movies APP

##  Technology used

- ReactJS
- Javascript ES6
- HTML5
- SCSS
- NodeJS
- MySQL
- Express
- Reactstrap

## API 

- get **localhost:5000/api/movies**,
Get all movies

- get **localhost:5000/api/search**,
Search a movie

- post **localhost:5000/api/new**,
Create one movie

- delete **localhost:5000/api/delete/:id**,
Delete a movie
